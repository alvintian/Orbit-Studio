# Free Freelance Website (Static)

This is a simple one-page website you can host for free.

## Quick edit (2 minutes)
Open `index.html` and replace:
- **Your Name**
- `youremail@example.com`
- Social links + project placeholders

## Run locally
Just double-click `index.html` or use a simple local server.

## Host for free
### Option A: GitHub Pages
1) Create a GitHub repo
2) Upload these files to the repo root
3) Repo **Settings → Pages → Source: Deploy from a branch → Branch: main → /(root) → Save**
4) Your site will be at: `https://YOUR-USERNAME.github.io/REPO-NAME/`

### Option B: Netlify
Drag-and-drop the folder into Netlify (or connect GitHub).

### Option C: Cloudflare Pages
Connect the GitHub repo and deploy as a static site.

Done.
